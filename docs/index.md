# fluxopt-plot

Interactive [plotly](https://plotly.com/python/) visualization for
[fluxopt](https://fbumann.github.io/fluxopt/) energy system models.

!!! warning "Early development"
    This package is experimental — the API may change between releases.

## Installation

```bash
pip install fluxopt-plot
```

## Quick start

```python
from fluxopt_plot import PlotAccessor

# After solving a fluxopt model:

```
